I created this page because I like streaks. In Singapore, we receive daily WhatsApp messages and it can be hard to tell how many days we've gone without a case, so I created this page so I could have an easy place to see that information. All data is sourced from the Ministry of Health COVID-19 site.

Some endpoints you can access:

  • https://lastlocalcase.sg/text - A text based response with only the number of days since the last case.
  • https://lastlocalcase.sg/json - A lightweight JSON response with the number of days since the last case and when the last case was detected.
  • https://lastlocalcase.sg/cases.json - The raw JSON data used to populate the data on the site.
